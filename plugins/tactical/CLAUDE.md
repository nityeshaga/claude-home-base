# Tactical Plugin

Structured workflows for thinking, creativity, decision-making, and operational work.

## Skills

| Skill | Use When |
|-------|----------|
| `lets-brainstorm` | User wants to brainstorm, think through an idea, explore a problem, or have a structured creative conversation. Runs a timed coaching session. |
| `chief-of-staff` | Executive assistant — manages email, maintains a running brief (living todo list), triages data sources, tracks action items across conversations and Slack, delivers formatted briefs. Learns preferences over time. |
| `are-you-sure` | Blind debiasing experiment — runs competing hypotheses through blind subagents when user makes a claim or asks "is X the best Y?". Listens to all sides, then compiles an honest answer. Works for tech, geopolitics, business, anything. |
| `prompt-engineer` | Expert prompt engineering for AI systems — writing, reviewing, or improving prompts for Claude, GPT, or other LLMs. Treats AI as a genius teammate who needs clear context, not tricks. |
| `investigate-yourself` | Forensic self-investigation — diagnose why Claudie behaved a certain way by tracing the full context pipeline (launcher, bot.py, session logs, CLAUDE.md, memory). Read-only diagnosis, no fixes. |
| `interview-me` | Timed discovery interview — asks questions one at a time to learn about the user and what they're trying to do. Default 5 minutes. |
| `help-me-write` | Help the user write an article or post by interviewing them to pull out insights, then polishing their draft without rewriting it. Keeps their voice intact. |
