# More AI Plugin

Other AI models for tasks they excel at — image generation, video creation, specialized reasoning.

## Skills

| Skill | Use When |
|-------|----------|
| `gemini-imagegen` | Quick/cheap image generation, exploration, style transfers, logos, stickers, product mockups, 4K output |
| `openai-imagegen` | Highest quality images, accurate text rendering, character consistency, client-facing assets, infographics |
| `gemini-thinking` | Complex multi-step reasoning tasks that benefit from Gemini's thinking approach |

## Image Generation: OpenAI vs Gemini

| Scenario | Use |
|----------|-----|
| Highest quality, client-facing assets | **OpenAI** |
| Accurate text in images | **OpenAI** |
| Multi-panel with character consistency | **OpenAI** |
| Quick/cheap generation, exploration | **Gemini** |
| Speed-critical (1-3s generation) | **Gemini** |
| 4K output | **Gemini** |
| Budget-sensitive bulk generation | **Gemini** |
